Static

staticis the default value. An element with the position static; is not positioned in any special way. A static element is said to be NOT postioned and an element with its position set to anything else is said to be positioned

Relative

Setting the top, bottom, left, and right properties of a relatively-positioned element will cause it to be adjusted away from its normal position. Other content will not be adjusted to fit into any gap left by the element.

fixed

A fixed element is positioned relative to the viewport, which means it always stays in the same place EVEN if the page is scrolled. As with relative, the top, right, bottom, and left properties are used.
absolute is the trickiest position value. Absolute behaves like fixed except relative to the nearest positioned ancestor instead of the relative viewport. If an absolutely positioned element has no positoned ancestors, it uses the document body, and stil moves along with page scrolling. Remember a "positoned" element is one whose positon is anything but static.
